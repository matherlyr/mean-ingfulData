Let's Talk About Data!


In chapter 6, Cairo talks about different ways to view certain data.


The Norm



This graph shows the average number of hours of sleep that college students get. From this histogram, we can see that the mean
median, and the mode are all around 7 hours of sleep. If we were to adjust the bin sizes on this graph, we could see that this set
of data is unimodal.


The Exceptions


When the same data of hours of sleep is graphed on a dot plot, we can see that there are outliers such as the point at 2 hours
of sleep. Therefore, we can most likely find an interesting story behind this outlier.
